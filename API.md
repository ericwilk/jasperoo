# Installation #
The url specified will always point to the latest release of jasperoo. To use an older version, download it from the [downloads page](http://code.google.com/p/jasperoo/downloads/list).
```
osgi start --url http://s.digitalface.ca/jasperoo-latest
```

# Removal #
Prior to installing a new version of jasperoo, remove the old one.
```
addon remove --bundleSymbolicName ca.digitalface.jasperoo
```
